# Suman Behera — Professional Data Science Portfolio

A dark, modern, responsive portfolio website for Suman Behera.

## Included

- Animated hero section
- Dark navy + cyan + purple theme
- Impact metrics
- Featured project cards
- Case-study modals
- Screenshot gallery
- Skills cards
- More repositories section
- Certificate links
- Resume download/view buttons
- Contact section
- Mobile responsive layout

## Main files

```text
index.html
style.css
script.js
assets/
  resume/
  screenshots/
  certificates/
```

## Deploy on GitHub Pages

1. Create a new repository, for example: `suman-portfolio`
2. Upload all files from this folder
3. Go to repository **Settings** → **Pages**
4. Select branch: `main`
5. Select folder: `/root`
6. Save
7. Your portfolio will be live at:
   `https://sumanbehera-ds.github.io/suman-portfolio/`
